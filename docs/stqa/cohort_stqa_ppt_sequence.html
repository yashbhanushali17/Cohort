System FlowSlide 6

System Flow Diagram

User -> React Frontend
Frontend -> Express Backend -> MongoDB
Frontend -> Socket.IO Server
Backend -> Gemini API

The user interacts with the frontend, the backend processes requests,
MongoDB stores persistent records, Socket.IO handles live updates,
and Gemini powers AI responses.
